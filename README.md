# Синхронизация

После завершения работы рекомендуется выполнить синхронизацию.

Во время синхронизации программа:

- скачивает изменения других переводчиков;
- загружает ваши локальные изменения;
- обновляет список переводов.

Если какая-либо строка уже была изменена на сервере, она автоматически станет **неутверждённой** и потребует повторной проверки.

### Скриншот

```
![Синхронизация](docs/images/sync.png)
```

---

# Отправка изменений

После проверки нажмите кнопку отправки.

На сервер отправляются **только утверждённые переводы**.

Неутверждённые изменения останутся в программе.

После успешной отправки утверждённые переводы автоматически применяются к локальным файлам, поэтому они перестают считаться изменёнными.

```
![Отправка](docs/images/upload.png)
```

# Рекомендуемый порядок работы

1. Запустите программу.
2. Выберите язык.
3. Выберите папку.
4. Введите никнейм.
5. Дождитесь загрузки данных.
6. Выберите файл.
7. Исправьте русский текст (при необходимости).
8. Добавьте или исправьте английский перевод.
9. Утвердите готовые строки.
10. Выполните синхронизацию.
11. Повторно проверьте утверждённые строки.
12. Отправьте изменения на сервер.
13. Закройте программу.

---

# Полезные советы

- Синхронизируйтесь как можно чаще.
- Перед отправкой ещё раз просматривайте утверждённые строки.
- При отсутствии Интернета регулярно сохраняйте изменения локально.
- После синхронизации некоторые строки могут снова стать неутверждёнными — это нормально, если другой переводчик успел изменить их раньше.
